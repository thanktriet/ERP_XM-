## Key Decisions Made

| Decision | Rationale | Alternatives |
|----------|-----------|--------------|
| `item_movements` riêng thay vì gộp vào `stock_movements` | Tránh phá vỡ spare_parts logic cũ, schema đơn giản hơn | Dùng chung bảng với `item_type` discriminator |
| `quantity` dương/âm trong `item_movements` | Phép tính tồn kho đơn giản hơn (chỉ `+= quantity`) | Dùng `movement_type` để quyết định dấu |
| `order_gifts` có trigger tự tạo `item_movements` | Tránh double-book, đảm bảo inventory nhất quán | Tạo thủ công từ controller |
| Pattern `DROP POLICY IF EXISTS` trước `CREATE POLICY` | PostgreSQL không hỗ trợ `CREATE POLICY IF NOT EXISTS` | Bọc trong `DO $$ BEGIN ... EXCEPTION ...` |
| `fix_accessories_and_gifts_v2.sql` thay vì v1 | Bảng `accessories` đã tồn tại với schema cũ (11 columns), cần ALTER | Xóa + tạo lại (nguy hiểm nếu có dữ liệu) |
